## What is "Spooky Action"?

This project is a playground for exploring the strange and counter-intuitive concepts of quantum mechanics through **logic games**.

The name "Spooky Action" is how Albert Einstein described **quantum entanglement**‚Äîthe idea that two particles can be connected instantaneously, no matter the distance.

This repository (and the games within it) aims to rescue fundamental concepts like this and transform them into challenging and educational game mechanics.

The goal of this project goes beyond simply using the word "quantum" as a superficial theme. The intention is to create games that teach or are metaphorically based on **how quantum computing actually works**.

Games here can focus on different layers of this stack:

* **Physical Layer (Hardware):** A game can simulate **decoherence** and the need for error correction.

* **Logical Layer (Firmware/Control):** A puzzle game focused on applying **quantum logic gates** (Hadamard, CNOT) to manipulate Qubits.

* **Algorithm Layer:** A strategy game about using **quantum parallelism** or **entanglement** to solve a problem faster than a classical computer.

* **Application Layer:** A game that abstracts all of this and simply uses "quantum logic" (e.g., an object being in two places at the same time) as its core mechanics.
